## Controls

| Key | Action |
| --- | --- |
| `[` / `]` | Switch providers |
| `Tab` / `Shift+Tab` | Switch providers when supported by the terminal |
| `Up` / `Down` or `k` / `j` | Switch dashboard sections |
| `Left` / `Right` | Select the previous or next table row |
| `q` or `Esc` | Quit |

## Local development

```bash
pnpm install
pnpm start
```
